## Installation

Ikuti Langkah-Langkah Berikut:

1. **Step 1: Update Composer**
   
   Run code berikut di Terminal:

   ```bash
   composer update
   ```
2. **Step 2: Generate Key Application**
   
   Run code berikut di Terminal:

   ```bash
   php artisan key:generate
   ```
3. **Step 3: Migrate**
   
   Run code berikut di Terminal:

   ```bash
   php artisan migrate
   ```
4. **Step 4: Run**
   
   Run code berikut di Terminal:

   ```bash
   php artisan serve
   ```
